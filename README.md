# 会社設立目的自動生成システム

会社設立時の事業目的を簡単に生成できるWebアプリケーションです。

## 特徴

- 🤖 **AI自動選定**: 業種・事業内容を入力するだけで、適切な事業目的を自動選定
- 📝 **編集可能**: 生成された目的は自由に編集・並び替え可能
- 📋 **一括コピー**: すべての目的をワンクリックでコピー
- 🎯 **23業種対応**: 情報通信業、飲食店、不動産、建設業など幅広い業種に対応

## 使い方

1. 業種や事業内容を入力（例: Web制作、飲食店、不動産仲介）
2. 「目的を自動生成する」ボタンをクリック
3. 生成された目的を確認・編集
4. 「すべての目的をコピー」で一括コピー

## 対応業種

- 飲食店・宿泊業
- 卸売・小売業
- 教育・学習支援業

- サービス業（マスコミ・出版・広告）
- 人材サービス
- サービス業（理容・美容・洗濯・生活）
- サービス業（芸能）
- 医療・福祉
- 運輸業・旅客運送業
- 金融・保険・投資業
- 電気・ガス・熱供給・水道業
- 不動産業
- 建設業
- 産廃業
- 製造業（金属・機械・プラスチック）
- 製造業（衣料品・家具）
- コンサルティング
